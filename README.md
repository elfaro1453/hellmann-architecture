## Architeture
MVVM with these layers:
* View
* ViewModel
* Data - Repositories
* Domain - UseCase

## Language
100% Kotlin

## Stack
* Retrofit
* Koin - Dependency Injection
* Coroutines
* Data binding
* Room - ORM  
* LiveData 
* Navigation library
* Unit tests / Medium tests  
* MockK  
* Espresso  

There is a branch using RxJava: rxjava
